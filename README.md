A Go project template with:
 - Unit test
 - WASM code

## Support
The following browsers are supported:
 - Google Chrome
 - Firefox
 - Safari

## Build
To build the WASM code, run `build_wasm`.
```
./build_wasm
```
And then you can open `index.html` on any supported browser.

To run tests, run `run_tests`.
```
./run_tests
```
